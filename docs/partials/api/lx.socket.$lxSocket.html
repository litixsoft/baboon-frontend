  • emit(eventName, data, callback)

    Send event with or without callback to socket server.

    Parameters
    ParamTypeDetails
    eventNamestring

    Name of server event

    data
    (optional)
    		object

    Data for server

    callback
    (optional)
    		function(err, success

    Callback function

    PropertyTypeDetails
    err
    (optional)
    		function

    Error callback function

    success
    (optional)
    		function

    Success callback function

    Returns
    function

    The socket.io emit function or an error function

    Example

    With callback: 

    $lxSocket.emit('name', data, function (err, success) {
    if (err) {
      console.log(err.status, err.message, err.stack);
    } else {
      console.log(success.status, success.data)
    }
  });
    Without callback: 
    $lxSocket.emit('name', data);
