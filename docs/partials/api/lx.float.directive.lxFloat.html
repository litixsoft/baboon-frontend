lxFloat
directive in module lx.float

Description

Convert and round any number to a float by given decimal places. It replaces a decimal comma with an decimal point. An non-float-input returns undefined.

For more information look at the guide.

Usage

as attribute
<input lx-float
     [lx-float="{number}"]>
   ...
</input>

Parameters

ParamTypeDetails
lx-float
(optional)
number

Number of decimal places.

(default: 2)
