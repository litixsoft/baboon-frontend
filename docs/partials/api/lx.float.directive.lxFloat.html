lxFloat
directive in module lx.float

Description

Convert and round any number to a float by given decimal places. It replaces a decimal comma with an decimal point. An non-float-input returns undefined. Optionally you can specify if the number should not be rounded.

For more information look at the guide.

Usage

as attribute
<input lx-float
     [lx-float="{number}"]
     [lx-float-round="{boolean}"]>
   ...
</input>

Parameters

ParamTypeDetails
lx-float
(optional)
number

Number of decimal places. (Default is 2)

(default: 2)

lx-float-round
(optional)
boolean

Specifies if the number should be rounded. (Default is true)

(default: true)
