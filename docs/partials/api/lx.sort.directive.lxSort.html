lxSort
directive in module lx.sort

Description

Directive for custom sorting or server side sorting.

For more details see our Guide.

Usage

as element:
<lx-sort
       field-name="{string}"
       sort-opts="{object}"
       on-sorting="{function(sortingOptions) }">
</lx-sort>

Parameters

ParamTypeDetails
fieldNamestring

The name of the property which should sort.

sortOptsobject

The options for configuration.

onSortingfunction(sortingOptions)

The callback which is called on sorting.
