lxComNavTree
directive in module lx.navigation

Description

Creates the navigation in a tree list.

For more information look at the guide.

Usage

as element:
<lx-com-nav-tree
       nav-linklist="{string}"
       [nav-template-path="{object}"]>
</lx-com-nav-tree>

Parameters

ParamTypeDetails
navLinkliststring

The list with all navigation links.

navTemplatePath
(optional)
object

The path to a template which should overwrite the standard layout.
