lxPager
directive in module lx.pager

Description

Creates an pager for custom or server side paging.

For more details see our Guide.

Usage

as element:
<lx-pager
       count="{number}"
       [[current-page]="{number}"]
       page-size="{number}"
       page-sizes="{Array}"
       on-paging="{function(pagingOptions) }">
</lx-pager>

Parameters

ParamTypeDetails
countnumber

The count of all items.

[currentPage]
(optional)
number

The current page to display.

pageSizenumber

The current page size.

pageSizesArray

An array of all page sizes which the user can select.

onPagingfunction(pagingOptions)

The callback which is called on paging.
